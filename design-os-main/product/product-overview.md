# Social Bounty

## Description
Social Bounty is a task marketplace where businesses post bounties with rewards and real people complete them with verifiable proof. It brings accountability, structured review, and transparent payout tracking to the gig-task economy.

## Problems & Solutions

### Problem 1: No Simple Way to Post Verifiable Tasks at Scale
Businesses need to outsource small, well-defined tasks but existing platforms are either too complex (freelance marketplaces) or too informal (social media posts). Social Bounty provides a clean bounty board where businesses create structured tasks with clear requirements, rewards, and deadlines.

### Problem 2: Proof-of-Completion Is Messy and Unstructured
When tasks are completed, there's no standardized way to verify the work was actually done. Social Bounty requires participants to submit structured proof (text, links, images, file uploads) tied directly to each bounty, giving businesses everything they need to evaluate in one place.

### Problem 3: Review and Approval Workflows Are Ad Hoc
Most task platforms lack a proper review pipeline, leading to disputes and confusion. Social Bounty implements a clear submission lifecycle (Submitted, In Review, Needs More Info, Approved, Rejected) so both sides always know where things stand.

### Problem 4: Payout Tracking Is Scattered Across Tools
Businesses lose track of who's been paid and who hasn't. Social Bounty provides built-in payout tracking (Not Paid, Pending, Paid) tied directly to approved submissions, keeping financials organized without requiring automated payment integrations.

### Problem 5: Platform Oversight Is an Afterthought
Most marketplaces lack proper admin tooling for troubleshooting and auditing. Social Bounty includes a super admin panel with user management, submission overrides with documented reasons, audit logging, and system health monitoring from day one.

## Key Features
- Bounty lifecycle management (Draft, Live, Paused, Closed)
- Structured proof-of-completion submissions with file uploads
- Multi-step review workflow with "Needs More Info" loop
- Manual payout tracking per approved submission
- Role-based access control (Participant, Business Admin, Super Admin)
- Organization management with member roles
- Full audit logging for all admin actions
- Super admin troubleshooting panel (user search, suspend/reinstate, overrides)
- Email notifications on status changes
- Simple reporting with counts and breakdowns
